# 🚀 Test Automation Portfolio

Projeto com foco em automação de testes utilizando diferentes ferramentas.

## 🔧 Tecnologias utilizadas

- Cypress (UI e API)
- Cucumber (BDD - Gherkin)
- Robot Framework (SeleniumLibrary)

## 📌 O que foi desenvolvido

### Cypress
- Testes E2E (navegação e validação de elementos)
- Testes de API com validação de status e resposta

### Cucumber
- Criação de cenários BDD utilizando Gherkin (Given/When/Then)

### Robot Framework
- Testes automatizados com SeleniumLibrary
- Abertura de navegador e validação de página

## ▶️ Como executar

Para rodar os testes, utilize os comandos abaixo:

### Robot Framework
py -m robot robot-project/test.robot

### Cypress  (modo interativo)
```bash
npx cypress open

npx cypress run
